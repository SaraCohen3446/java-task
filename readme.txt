מדריך הפעלה לפרוייקט
מה יש בפרוייקט -
- task-service-java 
 חלק צד שרת ב-Java לניהול רשימת משימות. מאפשר להוסיף, לעדכן, למחוק ולחפש משימות, לסמן משימות כבוצעות, ולהציג רשימה ממוינת. כל הנתונים נשמרים בקובץ JSON (data/tasks.json).
- part2-algorithms
  פונקציה שמקבלת מערך מספרים ומחזירה את כל תתי-הרצפים העולים מזהה רצפים ייחודיים ומדפיסה אותם מהארוך ביותר לקצר ביותר
- part3-design
  תכנון REST API ומחלקות למערכת ספרייה והזמנות/תשלומים. כולל מבני נתונים, פעולות עיקריות ואחריות כללית של המחלקות

הפעלה והרצה -
1.	פתחו את הפרוייקט ב- IntelliJ IDEA
2.	ודאו כי JDK מותקן ומוגדר כראוי ב-IDE
3.	להרצת הקוד הראשי:
   * עברו לקובץ Main.java  הרלוונטי נמצא בנתיבים
    \task-service-java\src\ ו- \part2-algorithms\src\
   * לחצו ימני Run  או השתמשו בקיצור הדרך להרצה ב IDE-
4.	צפו בפלט בקונסול של ה IDE-
